'TypeError':
  представляет собой ошибку, возникающую при недопустимом типе для переменной или параметра.

'SyntaxError':
  представляет собой ошибку, возникающую при разборе исходного кода или JSON.

'RangeError':
  представляет собой ошибку, возникающую при попытке передать параметром функции число, которое не входит в допустимый
    диапазон значений этого параметра функции.

'ReferenceError':
  Эта ошибка возникает, когда ссылка на переменную/элемент нарушена. То есть переменная/элемент не существует.
    const l=console.log
    const cat = "cat"
    cat
    dog // Переменной не существует > 'ReferenceError'

'URIError':
  Указывает на то, что одна из глобальных функций обработки URI была использована способом, несовместимым с ее определением.
  URI (Uniform Resource Indicator) в JS имеет функции: decodeURI, decodeURIComponent и т.д.

'EvalError':
  Используется для выявления ошибок при использовании глобальной функции eval().

'InternalError':
  Эта ошибка возникает внутри JS-движка, особенно когда ему нужно обработать слишком много данных и стек перерастает свой критический предел.
  Это происходит, когда JS-движок перегружен слишком большим количеством рекурсий, слишком большим количеством случаев переключения и т.д.
  Слишком много рекурсии, простой пример:
    function foo() {
      foo()
    }
    foo()
