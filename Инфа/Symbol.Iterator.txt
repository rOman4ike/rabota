Symbol.iterator
Два наиболее распространенных способа итерации массива в обратном направлении - это использование индексов (от array.length - 1 до 0) или Array.reverse, ИЛИ ЖЕ:
const reverse = arr => ({
  [Symbol.iterator]() {
    let i = arr.length;
    return {
      next: () => ({
        value: arr[--i],
        done: i < 0
      })
    }
  }
})
Вместо итерации в обратном направлении для заданного массива мы определили поведение итерации "назад" для любого массива, не изменяя порядок и не создавая клона исходного массива.
